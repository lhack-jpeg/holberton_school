Web Developer Bootcamp

This a fantastic course by Colt Steele that comprehensively covers the world of web development. It covers the basic of HTML, CSS and JS. It is a great course that also teaches you Node JS, Express and MongoDB.

Be sure to check out the course here.

SQL bootcamp

The SQL bootcamp is a detailed and indepth course that teaches the fundamentals of how to manipulate a SQL database. The course focuses on MySQL but the syntax learned from it can be applied easily to other SQL dbs. This course is provided by Colt Steele and is a fun way for complete newbies to gain serious knowledge on this powerful technology.

Be sure to check out the course here.

Scrapy: Web Scraping and Crawling

In this course provided by GoTrained Academy the user learns how to set and deploy scrapy to create powerful automated scripts for scraping large amounts of data from websites. This is a useful tool to gather data for populating databases at the start of an web application lifecycle.

Be sure to check out the course here.
