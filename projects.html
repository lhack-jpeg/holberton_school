Suite Selector

This is a hobby project created to help with the quality of life for staff and customers of Reece. The web application utilises NodeJS and Express as the backbone of the app. To seed the data of the application, I built my own web scraper using Scrapy and Zyte. The database is hosted on Mongo Atlas as a clone of my local database. To style the web application, I used SASS pre-processing. The website uses a form input to filter through the database and return the results of the form for what toilet will suit the current layout.
